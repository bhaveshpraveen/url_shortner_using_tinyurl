About:

1. This is a url_shortening program
2. It was built using python 3.5.2
3. It was tested on ubuntu 16.04
4. It requires internet connection
5. It uses urllib module

Instructions:

1. If you're running the program using the terminal add the links you want to shorten after you mention the python version and the filename
eg: python3 sample3.py link link link
(refer the pics )
 
2. if you don't want to add the links in the terminal you could add links in the input.txt file, one link in one line

(eg:
In input.txt add

 http://stackoverflow.com/questions/606191/convert-bytes-to-a-python-string
 https://www.youtube.com/watch?v=NwH0HvMI4EA
 https://www.youtube.com/watch?v=8BiOPBsXh0g
 )    
 Note: Please add the links along with https:// or http://
 
 Output:
 
 1. The output(s) is avaiable both in the terminal as well as in the file output.txt
